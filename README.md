## ⏱ TIMERS

จับเวลา Cooldown หลายกลุ่มพร้อมกัน

| ปุ่ม | การทำงาน |
|------|-----------|
| **+** | เพิ่ม timer ใหม่ |
| **🗑** | ลบ timer ที่หมดเวลาทั้งหมด |
| **×** | ลบ timer รายการนั้น |

**เพิ่ม Timer:**
1. กด **+**
2. ใส่ชื่อกลุ่ม
3. เลือกเวลา (5 / 10 / 15 / 20 นาที)
4. กด **▶ เริ่ม**

Progress bar เปลี่ยนสี 🟣 → 🟡 → 🔴 เมื่อใกล้หมดเวลา

## 🚫 BLACKLIST

จัดการรายชื่อผู้ถูก Blacklist

### เพิ่มรายชื่อ ( + )
1. กด **+**
2. ใส่ชื่อผู้เล่น
3. เลือกประเภท: **ประชาชน** หรือ **แก๊ง/ครอบครัว**
4. ใส่เหตุผลและค่าปรับ (IC)
5. กด **+ เพิ่ม BL**

### จัดการรายการ

| ปุ่ม | การทำงาน |
|------|-----------|
| `[สถานะ]` | คลิกเพื่อเปลี่ยนสถานะถัดไป |
| **✎** | แก้ไขหมายเหตุ |
| **×** | ปลด Blacklist |

**สถานะ:** รอดำเนินการ → ติดต่อแล้ว → รอจ่าย → ปฏิเสธ → จ่ายแล้ว

### ค่าปรับ ×2
รายการที่เกิน **20 นาที** และยังไม่จ่าย จะแสดงสี **แดง** พร้อม badge `×2 ⚠️`  
ยอดค่าปรับจะคำนวณเป็น 2 เท่าอัตโนมัติ

### ค้นหา 🔍
พิมพ์ชื่อในช่องค้นหาเพื่อกรองรายการ

## 🔄 การ Sync

- หน้าเว็บ **refresh อัตโนมัติ** ทุก 30 วินาที
- กด **⟳** เพื่อ refresh ทันที
- ข้อมูลจาก `announcer.py` จะ sync ขึ้นมาภายใน 15 วินาทีหลังมีการเปลี่ยนแปลง
- แก้ไขจากเว็บก็ sync กลับไปยัง `announcer.py` เช่นกัน
